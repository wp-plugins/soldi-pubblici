== Description ==

> Versione beta

Plugin per l'interfacciamento con il portale soldipubblici.gov.it

**This plugin is intended for italian public administrations only.**

= Shortcode =
Lo shortcode attualmente supportato è:

[wpgov-sp codiceente="000000000"]

sostituendo gli zeri con il codice ente della PA da valorizzare.

Di seguito tutti i parametri **in fase di sviluppo**:

`array(
	'anno' => '0', //0 = ultimo mese SIOPE
	'ricerca' => '',
	'tabella' => '',
	'codicecomparto' => 'PRO',
	'codiceente' => '',
	'chi' => '',
	'cosa' => ''
)`
